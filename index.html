Privacy Policy for Meta Ads Library Tool

Last Updated: 2025-11-24

1. Overview

This Privacy Policy describes how this application (“the App”) handles data when accessing Meta’s Ads Library API. The App retrieves publicly available ad transparency data only.

No personal data is collected, stored, or processed by this App.

2. Information We Collect

The App does not collect any personally identifiable information. The only data accessed comes directly from the Meta Ads Library API and includes:

  • Ad creative text and media information
  • Page name and advertiser information
  • Ad delivery start and stop dates
  • Ad spend ranges (if provided publicly by Meta)
  • Targeting categories (public transparency fields)

All data retrieved is public information provided by Meta for transparency purposes.

3. How We Use This Data

Public ad transparency data is used solely for:

  • Monitoring advertiser activity
  • Research and analysis
  • Reporting on marketing trends
  • Competitive intelligence

No user profiles are created, and no personal data is processed.

4. Data Sharing

We do not sell, share, or transfer any data to third parties. All data remains within the App and is used only for analysis.

5. Data Storage

No personal or private user data is stored. Any technical logs generated during operation contain no personal information and are automatically deleted periodically.
